# VTemplates

## Installation

```bash
mkdir -p ~/Library/Developer/Xcode/Templates/
cd ~/Library/Developer/Xcode/Templates/
git clone git@github.com:victorpereiradepaula/VTemplates.git
```

## Update

```bash
cd ~/Library/Developer/Xcode/Templates/
git pull
```

## Uninstallation

```bash
rm -rf ~/Library/Developer/Xcode/Templates/VTemplates
```
